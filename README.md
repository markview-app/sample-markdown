# MarkView Sample Markdown

This repository contains sample Markdown files to demonstrate the features and capabilities of the **MarkView** Chrome extension. You can use these samples to test how MarkView renders different Markdown elements and formats.

## How to Use

1. **Install MarkView**: If you haven't already, install the MarkView extension from the [Chrome Web Store](https://chrome.google.com/webstore/detail/cfopbpknalachedpcddhgbgjoigklien).
2. **Open Sample Files**: You can open any of the sample Markdown files in this repository using MarkView to see how they are rendered.

> [!NOTE]
> **Requirement**:
> Make sure to click the "RAW" button on GitHub to see the raw markdown content.
